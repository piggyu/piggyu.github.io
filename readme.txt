Overview:
    We are a platform that can:
        help you record and keep track of your body index
        offer a calories counter of your meals
        help you achieve the weight goal from the start point of dietary intake 
        visualize your body index and make your goal clearly visible



Run the project:

    Run in the draw6.py 

    main.py contains some of the user_profile, meal, and food OOP and the import of the food nutrients form

    Button.py stores the button OOP and textbox OOP

    User_information.css stores the user information from the user input

    Recommendation.py contains the backtracking algorithm of recommending meals based on the user target and present intake

    f.png and m.png are the user profile pictures



No libraries needs to be installed



Short commands:
    Press 'U' to uopdate the weight information
